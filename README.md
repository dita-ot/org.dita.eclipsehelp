Eclipse Help plug-in
===============

DITA-OT plug-in to generate Eclipse Help output.

Compatibility
-------------

-   DITA-OT 3.3

Installation
------------

Run the plug-in installation command:

```shell
dita --install org.dita.eclipsehelp
```
